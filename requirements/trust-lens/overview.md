# TrustLens

## コンセプト

ニュース記事やWeb情報の信頼度をAIでリアルタイム分析し、バイアスや情報源の質をスコアリングして可視化するアプリ。ユーザーが日常的に接する情報の「レンズ」として機能し、メディアリテラシーを自然に向上させる。

## ターゲットユーザー

- 日常的にニュースを読む20〜40代のビジネスパーソン
- 投資判断に正確な情報を求める個人投資家
- メディアリテラシー教育に関心のある教育関係者
- フェイクニュースの拡散を懸念するSNSユーザー

## 機能一覧

| ID | 機能名 | 概要 | 優先度 |
|----|--------|------|--------|
| 01 | 記事信頼度スコアリング | URLまたはテキストを入力すると信頼度スコアと根拠を表示 | 高 |
| 02 | バイアス分析 | 記事の政治的・商業的バイアスを検出しチャートで可視化 | 高 |
| 03 | ソーストラッキング | 情報源の引用チェーンを追跡し、一次情報源を特定 | 高 |
| 04 | パーソナルフィード | 信頼度フィルター付きのニュースアグリゲーター | 中 |
| 05 | 投資情報フィルター | 市場関連ニュースの信頼度と市場インパクトを分析 | 中 |
| 06 | ブラウザ拡張連携 | 閲覧中のページにリアルタイムで信頼度バッジを表示 | 低 |

## マネタイズ

フリーミアムモデルを採用:
- **無料プラン**: 1日10件の記事分析、基本的な信頼度スコア
- **Proプラン（月額980円）**: 無制限分析、詳細バイアスレポート、投資情報フィルター、ブラウザ拡張
- **Teamプラン（月額2,980円/5名）**: チーム内共有、API連携、カスタムフィルター

## 技術スタック（提案）

- **フロントエンド**: Next.js + TypeScript + Tailwind CSS
- **バックエンド**: Node.js + Hono（APIサーバー）
- **AI/ML**: Claude API（テキスト分析）、カスタム分類モデル（バイアス検出）
- **DB**: PostgreSQL（ユーザー・履歴）、Redis（キャッシュ・レートリミット）
- **インフラ**: Cloudflare Workers + D1（エッジ配信）
- **ブラウザ拡張**: Chrome Extension Manifest V3

## 運用方針

- Cloudflare Workersでサーバーレス運用、グローバルエッジ配信
- DBはCloudflare D1 + PostgreSQL（Neon）のハイブリッド
- CI/CD: GitHub Actions → Cloudflare Wrangler デプロイ
- 監視: Sentry（エラー）、Cloudflare Analytics（トラフィック）
- スケーリング: Cloudflare Workersの自動スケールに依存
