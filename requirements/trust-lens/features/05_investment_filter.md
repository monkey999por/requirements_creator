# 投資情報フィルター

## 概要

市場関連ニュースに特化した信頼度分析と市場インパクト予測を提供する。投資判断に影響するニュースの信頼性を検証し、センチメントと市場への影響度を分析することで、個人投資家のより良い意思決定を支援する。

## 画面構成

- **マーケットダッシュボード**: 主要指数のサマリーと関連ニュースの信頼度分布
- **ニュースカード**: 信頼度スコア + 市場インパクトスコア（高/中/低）+ センチメント（強気/中立/弱気）
- **銘柄別ビュー**: 特定銘柄に関連するニュースを信頼度順で表示
- **アラート設定画面**: 高インパクトニュースの通知条件設定

## ユーザーフロー

1. 「投資」タブを選択しマーケットダッシュボードを表示
2. 関心のある銘柄やセクターでフィルタリング
3. 各ニュースカードで信頼度とインパクトを確認
4. 気になるニュースをタップして詳細分析（ソーストラッキング含む）を閲覧
5. アラート設定で重要ニュースの通知を受け取る

## データモデル

- **market_news テーブル**: id, feed_item_id, market_impact_score, sentiment(bullish/neutral/bearish), related_tickers_json, sector
- **user_watchlist テーブル**: id, user_id, ticker, added_at
- **market_alerts テーブル**: id, user_id, conditions_json(min_impact, sentiment, tickers), enabled

## API設計

| メソッド | パス | 概要 |
|----------|------|------|
| GET | /api/market/feed | 市場ニュースフィードを取得（信頼度・インパクト付き） |
| GET | /api/market/ticker/:symbol | 銘柄別の関連ニュースを取得 |
| PUT | /api/market/watchlist | ウォッチリストの銘柄を更新 |
| PUT | /api/market/alerts | アラート設定を更新 |

## 非機能要件

- 市場ニュースの更新: 5分間隔
- インパクトスコアの算出は記事公開から30分以内に完了
- Proプラン限定機能
- 投資助言には該当しない旨の免責事項を表示
