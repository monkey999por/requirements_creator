# Digital Ethics Compass

## コンセプト

自分が日常的に利用するデジタルサービス・製品のエシカル度（企業倫理・プライバシー・社会的責任）をAIが自動スコアリングし、価値観に合った代替サービスへの乗り換えをパーソナライズ提案するアプリ。「自分のデジタルライフが世界にどんな影響を与えているか」を可視化し、意識的な選択を日常に組み込む。

## ターゲットユーザー

テクノロジーの恩恵を受けつつも、巨大テック企業の倫理的問題（プライバシー侵害、税逃れ、大量解雇、安全性軽視）に違和感を抱く20〜40代のデジタルネイティブ層。ボイコットほど極端な行動は取りたくないが、自分の消費行動をより倫理的な方向にシフトしたいと考えているユーザー。日常的にSNS・サブスク・ECなど多数のデジタルサービスを利用しており、「もっと良い選択肢があるなら知りたい」というニーズを持つ。

## 機能一覧

| ID | 機能名 | 概要 | 優先度 |
|----|--------|------|--------|
| 01 | サービス連携・利用状況スキャン | ユーザーが利用中のデジタルサービスを自動検出・手動登録し、利用状況を一覧化する | 高 |
| 02 | エシカルスコアリング | 企業ごとにプライバシー・労働・環境・税務・安全性の5軸でエシカルスコアを算出・表示する | 高 |
| 03 | 代替サービスレコメンド | エシカルスコアが低いサービスに対し、ユーザーの価値観と利用パターンに合った代替サービスを提案する | 高 |
| 04 | パーソナル倫理ダッシュボード | ユーザーのデジタルライフ全体のエシカル度を可視化し、改善推移をトラッキングするダッシュボード | 中 |
| 05 | 企業ニュースモニタリング | 利用中サービスの運営企業に関する倫理的ニュース（訴訟・データ漏洩・解雇等）をリアルタイムでアラート通知する | 中 |
| 06 | コミュニティレビュー | ユーザー同士が代替サービスの使用感や乗り換え体験をシェアするレビュー・ディスカッション機能 | 低 |

## マネタイズ

**フリーミアムモデル + B2B認証プログラム** の二本柱で収益化する。

- **フリーミアム（個人向け）**: 基本スコア閲覧・3サービスまでの登録は無料。プレミアム（月額580円）で無制限スキャン、詳細レポート、リアルタイムアラート、パーソナライズレコメンドを提供
- **B2B認証プログラム（企業向け）**: エシカルスコアが高い企業に「Ethics Verified」バッジを発行。認証取得・維持の年間ライセンス料（企業規模に応じた段階制）で収益化
- **アフィリエイト**: 代替サービスへの乗り換え時の紹介報酬（ユーザー体験を損なわない範囲で）

## 技術スタック（提案）

- **フロントエンド**: Next.js（App Router） + TypeScript + Tailwind CSS + shadcn/ui
- **バックエンド**: Next.js API Routes + tRPC（型安全なAPI通信）
- **データベース**: PostgreSQL（Supabase）+ Drizzle ORM
- **AI/ML**: OpenAI API（企業情報の要約・スコアリング根拠の生成）、Embedding検索（類似サービス推薦）
- **外部データ連携**: NewsAPI、企業情報API（OpenCorporates等）、OAuth連携（Google, Apple等のアカウント情報取得）
- **インフラ**: Vercel（フロントエンド + API）+ Supabase（DB + Auth）+ Upstash Redis（キャッシュ・レートリミット）
- **監視**: Sentry（エラー監視）+ Vercel Analytics（パフォーマンス）

## 運用方針

- **デプロイ**: Vercelへの自動デプロイ（GitHub連携、mainブランチマージで本番反映）。プレビュー環境でPRごとに動作確認
- **データ更新**: 企業エシカルスコアは日次バッチで最新ニュース・公開情報を反映。スコアリングモデルのバイアスを四半期ごとにレビュー
- **スケーリング**: VercelのEdge Functions + Supabaseのコネクションプーリングで初期は十分対応。ユーザー数10万超でCDN最適化・DB Read Replicaを検討
- **監視・アラート**: APIレスポンスタイム > 500ms、エラー率 > 1%でSlackアラート。スコアリングパイプラインの異常検知も実装
- **セキュリティ**: OAuth連携データの最小権限原則。ユーザーの利用サービスデータは暗号化保存。GDPR/個人情報保護法に準拠したデータ削除フロー
