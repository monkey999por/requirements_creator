# Value Compass（バリューコンパス）

## コンセプト

ユーザー固有の価値観（倫理性・環境配慮・コスト・品質・地域貢献など）を多軸でプロファイリングし、日常の意思決定（商品購入・サービス契約・投資先・メディア選択）においてAIが価値観適合度をスコアリング・代替提案するパーソナル意思決定支援アプリ。

## ターゲットユーザー

「自分の価値観に合った選択をしたいが、情報過多で判断に疲れている」20〜40代の意識的消費者。具体的には以下のような層を想定する:

- Big Tech依存を減らしたいが代替手段を探す時間がないテックワーカー
- エシカル消費に関心があるが商品ごとに調べるのが面倒な子育て世代
- ESG投資に興味があるが企業評価の基準がわからない個人投資家
- メディアリテラシーを高めたいが複数ソースを比較する余裕がない社会人

## 機能一覧

| ID | 機能名 | 概要 | 優先度 |
|----|--------|------|--------|
| 01 | 価値観プロファイリング | 対話形式の質問とシナリオ選択でユーザーの価値観を多軸で数値化・可視化する | 高 |
| 02 | 商品・サービス評価スキャン | バーコード/URL入力で商品・サービスの価値観適合スコアを即時表示する | 高 |
| 03 | 代替提案エンジン | 現在利用中のサービスや購入予定商品に対して、価値観により合致する代替候補をAIが提案する | 高 |
| 04 | メディアバイアスチェッカー | ニュース記事URLを入力すると、複数メディアの同一トピック報道と比較しバイアス傾向を可視化する | 中 |
| 05 | 投資ポートフォリオ価値観診断 | 保有資産や投資先企業を価値観軸（ESG・倫理性・社会貢献度）で評価し、改善提案を行う | 中 |
| 06 | 価値観コミュニティ | 近い価値観を持つユーザー同士が代替サービスや商品のレビューを共有できるコミュニティ機能 | 低 |
| 07 | 月次インパクトレポート | 1ヶ月間の選択履歴を集計し、価値観に沿った行動がどれだけできたかをレポートとして可視化する | 中 |

## マネタイズ

**フリーミアムモデル + B2Bライセンス の二軸構成:**

- **無料プラン**: 価値観プロファイリング、月10回までの商品スキャン、基本的な代替提案
- **プレミアムプラン（月額680円）**: 無制限スキャン、投資ポートフォリオ診断、メディアバイアスチェッカー、月次レポート、コミュニティ全機能
- **B2Bライセンス**: 企業がCSR/ESGスコアを自社商品に表示するためのAPI提供（従量課金）。企業はValue Compassの認証バッジを商品ページに掲載可能
- **アフィリエイト**: 代替提案で推薦した商品・サービスへの送客による成果報酬

## 技術スタック（提案）

- **フロントエンド**: Next.js (App Router) + TypeScript + Tailwind CSS + Recharts（価値観レーダーチャート等）
- **バックエンド**: Next.js API Routes + Prisma ORM
- **AI/ML**: OpenAI API（価値観分析・代替提案生成）、Embedding APIによる商品/サービスのベクトル類似度検索
- **データベース**: PostgreSQL（ユーザー・プロファイル・評価データ）+ pgvector（ベクトル検索）
- **外部API連携**: Open Food Facts（食品データ）、NewsAPI（メディア比較）、Yahoo Finance API（投資データ）、バーコードルックアップAPI
- **認証**: NextAuth.js（Google/Apple/Email）
- **インフラ**: Vercel（フロントエンド・API）+ Supabase（PostgreSQL + pgvector）
- **キャッシュ**: Upstash Redis（APIレスポンスキャッシュ、レート制限）

## 運用方針

- **インフラ**: Vercelによるサーバーレスデプロイ。SupabaseのマネージドpostgresqlでDB運用コストを最小化
- **デプロイ**: GitHub連携によるプレビューデプロイ + mainブランチへのマージで本番自動デプロイ
- **監視**: Vercel Analytics + Sentry（エラー監視）+ Upstash Redis Insights（キャッシュヒット率）
- **スケーリング**: Vercelのエッジファンクションによる自動スケール。DBはSupabaseのコネクションプーリングで対応
- **データ更新**: 企業評価データ・商品データは日次バッチで外部APIから取得・更新。ユーザーからのフィードバックによるスコア補正は即時反映
- **バックアップ**: Supabaseの自動バックアップ（日次）+ Point-in-Time Recovery
- **セキュリティ**: 個人の価値観データは機密性が高いため、暗号化保存・最小権限アクセスを徹底。GDPR/個人情報保護法に準拠したデータ管理
